# LensArt — Photography Portfolio Marketplace

LensArt is a vibrant, modern client-side photography portfolio and hiring platform. Discover world-class photographer portfolios, interact with live galleries, track stats in real-time, and hire artists for your events.

## ✨ Features
- **Vibrant Light Theme**: A clean, premium slate-and-indigo bright aesthetic with subtle transitions and glassmorphism.
- **Realistic Portraits**: Handcrafted professional photographer avatars matching their artistic portfolios.
- **Dynamic Portfolios**: Registered photographers get a dedicated profile template (`photographer/profile.html?id=usr_xxx`) displaying bios, locations, hourly rates, and portfolio showcase images.
- **Interactive Booking Module**: Secure guest-friendly commission form allowing clients to book sessions with budget, date, and project specifications.
- **Photographer Marketplace Directory**: Real-time listing of all available photographers directly on the homepage.
- **Role-Based Dashboards**:
  - **Photographers**: Manage showcase portfolios, edit professional details, review and Accept/Decline client job requests, and track dashboard views.
  - **Clients**: Capture webcam photos, review sent booking requests, and manage followed artists.
- **Real-Time Interactive Gallery**: Open photos in a premium lightbox overlay that tracks view counts and lets users toggle likes in real-time via `localStorage`.

## 🛠️ Technologies Used
- HTML5 (Semantic Layouts)
- CSS3 (Vanilla Variables & Responsive Grid Systems)
- Javascript (ES6 Modules & Local Database Persistence)

## 🚀 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Jayadeep7869/photography-portfolio.git
   ```
2. Open `index.html` directly in any web browser, or serve it using a local server:
   ```bash
   npx live-server
   ```
